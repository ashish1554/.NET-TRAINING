## Access modifiers


**Answer:**  

### 1)Public
Accesible from same class,same assembly,derived class of different assembly,derived class of same assembly

### 2)Private
Accesible within the same  class only

### 3)Protected
Accesible from the same class,derived class of the same assembly as well as different assembly 

### 4)Internal
Accesible within  the entire assembly and the  same class and derived class of the same assembly

### 5)Protected Internal
It is OR of the protected and internal means accesible from places where internal is accesible or internal accesible

### 6) Private Protected
Only accesible within the derived class of the same assembly and within the same class
